# Overview

{Important! Do not say in this section that this is college assignment. Talk about what you are trying to accomplish as a software engineer to further your learning.}

MY software is a drawing app made from kotlin. It uses tool such as swing, JDK, and awt to let the user draw and create shapes
on a canvas. It uses several files to store data, decide where parts are drawn, what parts are drawn, and create a window for
the user to draw and decide what to use.

My pupose of creating this app was to make a software that allowed the user draw lines and shapes, and decide what tools they
want to use and to clear the app if they wanted to restart.

# Development Environment

Tools I used are IntelliJ IDEA for my code editor as it runs kotlin better than vs code, I used kotlin to make the app, 
and Java development kit to run swing applications.

The language I used was kotlin which is  language normally used for androids. Kotlin it a language that runs on JVM and is able to use a lot 
of java libraries. The libraries I used was swing for creating the gui and sections withing, awt for coloring, borders and drawing,
mouse events, and then standard kotlin bult ins for collections.
